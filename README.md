# 📝 Custom PSR Tool (Problem Steps Recorder Alternative)

A lightweight, portable tool for automatically documenting workflows and clicks on your PC. 
With Microsoft phasing out the classic "Problem Steps Recorder" (PSR) in Windows, this tool offers the perfect, modern alternative – exporting your steps directly into an editable Word document!

Developed by **Sacha Schwendimann**.

## ✨ Key Features

* **Word Export instead of HTML:** Every click is saved live into a clean `.docx` document. No data loss if the system crashes!
* **Multi-Monitor Support:** The tool automatically detects which screen you clicked on and only captures that specific monitor.
* **Smart Click Markers:** Clicks are highlighted directly on the screenshot (🔴 Red = Left click, 🟠 Orange = Right click) along with a visible mouse cursor.
* **Window Spy:** Automatically detects and logs the active application (e.g., "Microsoft Word" or "Google Chrome").
* **Detail Magnifier (Optional):** Automatically inserts a zoomed-in crop of the exact click area next to the full screenshot.
* **Session Management:** Automatically creates organized folders stamped with the current date and time for each new guide.
* **Bilingual:** The user interface can be switched between English and German with a single click.

## 🚀 How to Use

1. Download the `mein_psr.exe` from the Releases tab (no installation required).
2. Launch the program. The small control panel will always stay on top of your windows.
3. Click on **"▶️ New Recording"**.
4. Perform your workflow. The tool runs in the background and documents everything.
5. Click on **"💾 Finish Recording"**. 
6. Done! Your finished Word document is now waiting for you in the automatically generated folder `Meine_Aufzeichnung`.

# 📝 Custom PSR Tool (Problem Steps Recorder Alternative)

Ein leichtgewichtiges, portables Tool zur automatischen Dokumentation von Arbeitsschritten und Klicks am PC. 
Da Microsoft den klassischen "Problem Steps Recorder" (PSR) in Windows auslaufen lässt, bietet dieses Tool die perfekte, moderne Alternative – direkt exportiert als editierbares Word-Dokument!

Entwickelt von **Sacha Schwendimann**.

## ✨ Hauptfunktionen

* **Word-Export statt HTML:** Jeder Klick wird live in ein sauberes `.docx` Dokument geschrieben. Kein Datenverlust bei Abstürzen!
* **Multi-Monitor-Support:** Das Tool erkennt automatisch, auf welchem Bildschirm geklickt wurde, und fotografiert nur diesen Monitor.
* **Intelligente Klick-Marker:** Klicks werden direkt auf dem Screenshot markiert (🔴 Rot = Linksklick, 🟠 Orange = Rechtsklick) und mit einem künstlichen Mauszeiger versehen.
* **Fenster-Spion:** Das aktive Programm (z. B. "Microsoft Word" oder "Google Chrome") wird erkannt und im Dokument notiert.
* **Detail-Lupe (Optional):** Neben dem großen Screenshot wird automatisch ein herangezoomter Ausschnitt (Lupe) des Klickbereichs eingefügt.
* **Session-Verwaltung:** Automatische Erstellung von Ordnern mit Datum und Uhrzeit für jede neue Anleitung.
* **Zweisprachig:** Die Benutzeroberfläche kann per Knopfdruck zwischen Deutsch und Englisch umgeschaltet werden.

## 🚀 Nutzung

1. Lade dir die `mein_psr.exe` aus den Releases herunter (keine Installation notwendig).
2. Starte das Programm. Das kleine Menü bleibt immer im Vordergrund.
3. Klicke auf **"▶️ Neue Aufzeichnung"**.
4. Führe deine Arbeitsschritte aus. Das Tool läuft im Hintergrund und dokumentiert alles.
5. Klicke auf **"💾 Anleitung abschließen"**. 
6. Fertig! Dein fertiges Word-Dokument liegt nun im automatisch erstellten Ordner `Meine_Aufzeichnung`.
